When you change gl.POINTS to gl.LINES it will draw a line between every set of points so a line will be drawn betweem points 1 and 2 then 3 and 4 and so on. If you change it to gl.LINE_STRIP it will connect all of the circle dots except the first and last ones which will not connect directly to each other. If you change it to gl.LINE_LOOP it connects all points and creates a loop.
